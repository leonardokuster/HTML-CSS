Principais formatações:

Negrito / Destaque

Nesta frase, temos um termo em negrito utilizando a tag <b>, que é uma tag não semântica, ou seja, não tem significado, apenas forma.

Nesta outra frase, temos um termo em destaque utilizando a tag <strong>, que é semântica.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> que não é semântico.

Nesta outra frase, temos um termo em ênfase usando a tag <em>, que é semântica.

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

OBS: esse style que utilizei dentro da tag não se enquadra mais em HTML e sim CSS, porque se não fosse alterar a cor, ficaria na cor padrão que é amarelo.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído, para indicar que ele deve ser lido mas não considerado.

Texto inserido

Podemos marcar um texto como inserido, para dar uma ênfase em indicar que ele foi adicionado depois.

Existe também o sublinhado com tag <u>, que não é semântica.
